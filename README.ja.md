テクノロジーと言語に興味があるソフトウェア開発者です。現在は、**プログラミング**と**機械学習（Machine Learning）**を融合させ、日本語学習をサポートするツールの開発に挑戦しています。

### 自己紹介
- **現在の目標:** 漢字学習に特化したツールを構築するための機械学習システムの開発。
- **使用技術:** フロントエンド（モバイル）はFlutter、バックエンドはNode.jsを採用しています。
- **趣味:** プログラミングと日本文化。

### 主要プロジェクト
* **[kanji-app](https://github.com):** 漢字の定着とスマートな練習を可能にするために、**Flutter**で構築しているモバイルアプリケーションです。
